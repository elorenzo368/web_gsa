# GSA Web

Aplicación web básica construida con Vite y React.

## Requisitos

- Node.js 22.14.0

## Instalación

```bash
npm install
```

## Desarrollo

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Rutas

- `/` - Página principal (index)
- `/citrus` - Página Citrus

Ambas rutas muestran actualmente una imagen de "Sitio en construcción".

## Build

Para crear una build de producción:

```bash
npm run build
```

Para previsualizar la build:

```bash
npm run preview
```
